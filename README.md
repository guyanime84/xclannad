# xclannad
Xclannad is a simple and lightweight way to run the Clannad PC (HD Edition) visual novel directly on Android devices. It brings the full desktop experience of Clannad to your phone, preserving all visual, audio, and gameplay quality with easy setup.


1. Install the Xclannad APK
Download and install the xclannad.apk file on your Android device.
Allow installation from unknown sources if prompted.

2. Copy the Game Files
From your PC version of Clannad HD Edition, copy all game files to your phoneâ€™s internal storage under:
/storage/emulated/0/clannad/

3. Apply the Patch
Extract the provided patch archive.
Paste (and replace) its contents into the same /clannad folder containing the game files.

4. Launch the Game
Open the installed Xclannad app.
The game should start automatically and load your copied files.
